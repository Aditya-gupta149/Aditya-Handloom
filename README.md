# 🛍️ Aditya Handloom

A modern full-stack e-commerce platform built with the MERN stack, designed to deliver a smooth shopping experience for premium handloom and fashion products.

🌐 Live Demo: https://aditya-handloom.vercel.app

---

## ✨ Features

### 👤 Authentication

- User registration and login
- JWT-based authentication
- Secure password encryption
- Role-based access control (Admin/User)

### 🛒 Shopping Experience

- Browse products
- Product search and filtering
- Product categories
- Shopping cart functionality
- Checkout flow
- Stripe payment integration

### 📦 Orders

- Place orders securely
- View order history
- Order status updates
- Real-time stock management

### 🛠️ Admin Dashboard

- Add new products
- Edit product details
- Delete products
- Manage users
- Manage orders
- Update order status

---

## 🚀 Tech Stack

### Frontend

- React.js
- Vite
- Tailwind CSS
- React Router
- Axios

### Backend

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- bcrypt.js

### Payments & Deployment

- Stripe
- Vercel
- Render
- MongoDB Atlas

## ⚙️ Environment Variables

### Backend (`server/.env`)

```env
PORT=5000
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_secret_key
STRIPE_SECRET_KEY=your_stripe_secret_key
CLIENT_URL=https://aditya-handloom.vercel.app
```

### Frontend (`client/.env`)

```env
VITE_API_URL=https://your-render-backend-url/api
VITE_STRIPE_PUBLIC_KEY=your_public_key
```

## 🖥️ Installation

### Clone the repository

```bash
git clone https://github.com/Aditya-gupta149/Aditya-Handloom.git
```

### Backend setup

```bash
cd server
npm install
npm run dev
```

### Frontend setup

```bash
cd client
npm install
npm run dev
```
